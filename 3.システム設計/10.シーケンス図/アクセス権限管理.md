```mermaid
sequenceDiagram
  participant ユーザー
  participant フロントエンド
  participant バックエンド
  participant データベース

  loop ユーザー登録フロー
    ユーザー->>フロントエンド: ユーザー登録画面表示要求
    フロントエンド->>ユーザー: ユーザー登録画面表示
    ユーザー->>フロントエンド: ユーザーID, パスワード, 権限を入力
    フロントエンド->>バックエンド: ユーザー登録APIリクエスト (POST /api/users)
    バックエンド->>データベース: ユーザーアカウント登録
    alt 登録成功
      データベース-->>バックエンド: 登録されたユーザー情報
      バックエンド-->>フロントエンド: 登録成功レスポンス
      フロントエンド->>ユーザー: 登録完了メッセージ表示
    else 登録失敗
      データベース-->>バックエンド: エラー情報
      バックエンド-->>フロントエンド: エラーレスポンス
      フロントエンド->>ユーザー: エラーメッセージ表示
    end
  end

  loop ユーザー権限設定フロー
    ユーザー->>フロントエンド: ユーザー権限設定画面表示要求
    フロントエンド->>ユーザー: ユーザー権限設定画面表示
    ユーザー->>フロントエンド: ユーザーID, 権限を選択
    フロントエンド->>バックエンド: ユーザー権限設定APIリクエスト (PUT /api/users/{user_id}/permissions)
    バックエンド->>データベース: ユーザー権限更新
    alt 設定成功
      データベース-->>バックエンド: 更新されたユーザー情報
      バックエンド-->>フロントエンド: 設定成功レスポンス
      フロントエンド->>ユーザー: 設定完了メッセージ表示
    else 設定失敗
      データベース-->>バックエンド: エラー情報
      バックエンド-->>フロントエンド: エラーレスポンス
      フロントエンド->>ユーザー: エラーメッセージ表示
    end
  end

  loop 操作ログ記録フロー
    ユーザー->>フロントエンド: 操作実行
    フロントエンド->>バックエンド: 操作ログ記録APIリクエスト (POST /api/operation-logs)
    バックエンド->>データベース: 操作ログ記録
    データベース-->>バックエンド: 記録成功
    バックエンド-->>フロントエンド: 記録成功レスポンス
  end

  loop 操作ログ検索フロー
    ユーザー->>フロントエンド: 操作ログ検索画面表示要求
    フロントエンド->>ユーザー: 操作ログ検索画面表示
    ユーザー->>フロントエンド: 検索条件（ユーザーID、期間、操作の種類）を入力
    フロントエンド->>バックエンド: 操作ログ検索APIリクエスト (GET /api/operation-logs)
    バックエンド->>データベース: 操作ログ検索
    alt 検索成功
      データベース-->>バックエンド: 検索結果
      バックエンド-->>フロントエンド: 検索結果表示
      フロントエンド->>ユーザー: 検索結果表示
    else 検索失敗
      データベース-->>バックエンド: エラー情報
      バックエンド-->>フロントエンド: エラーレスポンス
      フロントエンド->>ユーザー: エラーメッセージ表示
    end
  end
