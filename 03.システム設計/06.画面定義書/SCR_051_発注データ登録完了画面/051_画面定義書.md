# 発注データ登録完了画面

## 基本情報
| 項目 | 内容 |
|------|------|
| ID | SCR-051 |
| 分類 | 発注管理 |
| 概要 | 発注データの登録が完了したことを知らせる画面。 |

## 構成要素
完了メッセージ、発注状況照会画面へのリンク、発注書自動生成画面へのリンク

## 操作手順
1. 完了メッセージを確認。
2. 発注状況照会画面へ遷移するか、発注書自動生成画面へ遷移。
