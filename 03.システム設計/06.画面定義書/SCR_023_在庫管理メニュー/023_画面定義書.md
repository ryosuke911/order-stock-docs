# 在庫管理メニュー

## 基本情報
| 項目 | 内容 |
|------|------|
| ID | SCR-023 |
| 分類 | 在庫管理 |
| 概要 | 入庫、出庫、在庫移動、在庫調整の各登録画面への入り口となるメニュー画面 |

## 構成要素
メニューボタン（入庫登録、出庫登録、在庫移動登録、在庫調整登録）

## 操作手順
各機能へのボタンをクリックして遷移

## アクセス権限
| 項目 | 内容 |
|------|------|
| 利用者 | 在庫管理担当者 |
| 権限 | loged-in user |
